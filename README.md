# Racetrack RL – PPO

Bu proje, racetrack ortamında şerit takibi öğrenen bir ajanı 
PPO algoritması ile eğitmeyi amaçlamaktadır.

## Özellikler
- PPO (Proximal Policy Optimization)
- Actor-Critic mimarisi
- Discrete action space (yalnızca direksiyon)
- Custom pygame render
- Sprite tabanlı araç görselleştirme

## Çalıştırma

### Eğitim
python train.py

### Demo
python custom_play.py
